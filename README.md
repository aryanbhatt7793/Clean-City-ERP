**CleanCity**: Data-Backed Waste Reporting & Circular Economy Platform
CleanCity is a web-based waste management system designed to detect, report, and manage unidentified garbage dumping locations while promoting structured household waste segregation and recycling-based incentives.

**Features**
📍 Citizens upload dumping site images and manually enter location details
✅ Zone-based verification to filter fake/duplicate reports
♻️ Segregation drives with NGOs or waste pickers
🏭 Zone collection center (“Darkhouse”) for recyclable storage
💰 Recycling marketplace with bidding system
📢 Municipal escalation for non-recyclable waste
🏠 Household segregation model with reward coins
📊 Data-driven risk scoring to predict future dumping hotspots
🛒 Marketplace for recycled & sustainable products

**Tech Stack**

**Frontend:** HTML, CSS, JavaScript
**Backend:** PHP
**Database:** MySQL
**Server:** Apache (XAMPP)

**Architecture Overview**

Client-side UI built with pure HTML/CSS/JS
PHP handles API requests & business logic
MySQL stores user reports, zone data, reward coins, and marketplace records
Apache server hosts the application locally (XAMPP)
